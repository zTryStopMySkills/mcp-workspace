## IMÁGENES — Lista de lo que necesitamos

### Prioridad ALTA
- [ ] 3-5 fotos del local / fachada (exterior e interior)
- [ ] 1 foto "estrella" para galería principal (horizontal, buena luz)
- [ ] Foto de los productos más vendidos

### Prioridad MEDIA
- [ ] Fotos de ambiente / sesiones de shisha
- [ ] Fotos de vapers y mods más vendidos
- [ ] Foto del fundador/equipo (opcional pero humaniza el negocio)

### Formato recomendado
- Horizontal (16:9 o similar) para galería
- Cuadrada (1:1) también válida
- Mínimo: 800×600 píxeles
- Formato: JPG o PNG

### Cómo enviar las fotos
**Opción 1:** Compartir carpeta de Google Drive o iCloud
**Opción 2:** Enviar por WhatsApp (en calidad original, no comprimida)
**Opción 3:** Subir a https://imgbb.com y enviar los enlaces
